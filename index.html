HOW CSS GRADIENTS WORK???

So without wasting any more time let's get started. Shall we???

There are basically 2 types of gradients.

  1. Linear Gradient
  2. Radial Gradient
Linear Gradient applies the 2 colors in linear/sequential way. While the Radial Gradient applies the colors in circular/elliptical way. For more information try to change the colors for linear as well as radial gradient in the output and see what happens. Believe me its going to be very much fun which you'll like and enjoy.
